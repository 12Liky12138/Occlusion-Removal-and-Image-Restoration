## Requirements

* python: 3.7

* pytorch>=0.4.1

* 下载pycocotools:
   
   ```shell
   pip install "git+https://github.com/cocodataset/cocoapi.git#subdirectory=PythonAPI"
   ```

* 下载需要的依赖:

    ```shell
    pip install -r requirements.txt
    ```
